# Unit 1 ¬∑ Day 1 ‚Äî Project Setup & First Server

## Key concepts

**Node.js** ‚Äî JavaScript running on your machine (not in a browser). Built on Chrome's V8 engine. Single-threaded, but handles I/O (network, files) asynchronously via libuv.

**npm** ‚Äî downloads libraries. `package.json` is the project manifest.
- `dependencies` ‚Üí needed at runtime (Express)
- `devDependencies` ‚Üí needed only during development (TypeScript, ts-node-dev)

**TypeScript** ‚Äî adds types to JavaScript. You write `.ts` ‚Üí compiler (`tsc`) outputs plain `.js` ‚Üí Node runs the `.js`. TypeScript itself never runs directly.

**Express** ‚Äî a thin wrapper around Node's built-in `http` module. Handles routing, middleware, and JSON parsing without boilerplate.

**CommonJS vs ES Modules** ‚Äî you write `import` in TypeScript, but `tsconfig.json` compiles it to `require()` (CommonJS) so Node can run it. Just use `import` syntax, the compiler handles the rest.

---

## Watch out for

- `@types/express` and `@types/node` are required ‚Äî TypeScript can't infer types for third-party packages without them
- Never edit files in `dist/` ‚Äî they're generated by `tsc` and overwritten every build
- Port already in use ‚Üí `lsof -ti:3000 | xargs kill`

## Steps

- [ ] 1. Create folders: `tennis-log/backend/src` and `tennis-log/frontend`
- [ ] 2. `cd tennis-log/backend` ‚Üí `npm init -y`
- [ ] 3. Install runtime dep: `npm install express`
- [ ] 4. Install dev deps: `npm install -D typescript @types/node @types/express ts-node-dev`
- [ ] 5. Create `tsconfig.json` (see below)
- [ ] 6. Create `src/index.ts` (see below)
- [ ] 7. Add scripts to `package.json` (see below)
- [ ] 8. Run `npm run dev` ‚Üí open `http://localhost:3000/health` in browser

## Code to write

**`tsconfig.json`** (in `backend/`)
```json
{
  "compilerOptions": {
    "target": "ES2020",
    "module": "commonjs",
    "outDir": "./dist",
    "rootDir": "./src",
    "strict": true,
    "esModuleInterop": true
  }
}
```

**`src/index.ts`**
```ts
import express from 'express'

const app = express()
const PORT = 3000

app.get('/health', (req, res) => {
  res.json({ status: 'ok' })
})

app.listen(PORT, () => {
  console.log(`Server running on http://localhost:${PORT}`)
})
```

**`package.json` scripts**
```json
"scripts": {
  "dev": "ts-node-dev --respawn src/index.ts",
  "build": "tsc",
  "start": "node dist/index.js"
}
```
